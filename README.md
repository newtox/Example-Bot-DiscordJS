# Example Bot DiscordJS
> A discord bot example using discord.js

## Installation

```
npm install
```

## Usage

Fill out the values in the config/config.js file.

```javascript
const PREFIX = '';
const TOKEN = '';
const OWNER = '';
```

Start the bot using **nodemon** in console.
